INSTALAÇÃO
------------------------------------

Copie a pasta 'lucastds02' para a pasta 'themes' do XOOPS e configure
este tema como padrão e/ou selecionável através das pereferências no
painel de administração do XOOPS.


INSTRUÇÕES
------------------------------------

- É possível personalizar o logotipo exibido no topo das páginas
  substituindo-se o arquivo 'logo.gif' pela imagem desejada. Você poderá
  usar o arquivo 'logo_blank.gif' como base para criar seu próprio
  logotipo.

- Para mudar a largura das páginas, abra o arquivo 'style.css' e
  localize a seguinte linha:
      #maintable {width: 760px; margin: auto;}
  Troque '760px' pela largura desejada.
      Exemplos: 100%, 80%, 700px, 800px.

- Para mudar o idioma das mensagens de boas vindas para português,
  substitua o arquivo 'theme.html' pelo 'theme_portuguesebr.html',
  localizado na pasta 'extras'.
